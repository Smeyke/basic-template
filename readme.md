#basic-template

------

Hello there.

This is my basic template to start projects.

I work with sass and have my .sass files scructured like this:

```
file/
├── index.html
├── css/
│   ├── style.css

└── sass/
    └── style.sass
    └── file_01.sass
    └── file_02.sass
    └── file_03.sass

```


Have fun!
